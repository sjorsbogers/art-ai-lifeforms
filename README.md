# ART — AI Lifeforms

A series of browser-based AI art experiments exploring machine identity, embodiment, and self-discovery.

Inspired by Cyrus Clarke's *I Gave an AI a Body* and the MIT Media Lab **inFORM** shape display — a 30×30 grid of 900 physical actuating pins where an AI was asked to discover its own identity through physical form. Nobody told it what to do.

---

## Lifeforms

| # | Name | Description | Stack |
|---|------|-------------|-------|
| 01 | Pin Grid | AI discovers identity through a 30×30 actuating pin grid | Vanilla JS + Three.js |

---

## Running a lifeform

Open the lifeform's `index.html` in Chrome or Firefox. No install required.
